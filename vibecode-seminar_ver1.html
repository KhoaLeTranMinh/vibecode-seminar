Methodology Comparison
01 · Theory
14 / 37
Software Methodologies
Waterfall vs
Agile.
Traditional · Waterfall
Sequential & Rigid
Full planning upfront. Each phase must finish before the next begins. Widely used by Japanese enterprises. Changes mid-project are expensive.
Modern · Agile / Scrum
Iterative & Flexible
Work in short sprints. Adapt to changes quickly. Build incrementally — test early, improve often. The dominant approach in most software teams today.

What is Vibe Coding?
Describe It.
AI Builds It.
You describe what you want in plain English. The AI writes the code. Your job: guide, test, and refine.
GitHub Copilot Agent · Natural Language → Code
Why
"Vibe"?
You code based on intent, not syntax.
Instead of getting bogged down in brackets and semicolons, you express the vibe of what you want. Your role shifts from typing code to directing the AI.
