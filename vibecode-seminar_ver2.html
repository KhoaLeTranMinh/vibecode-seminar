Machine Learning — Three Ways to Learn
Supervised,
Unsupervised &
Self-Supervised.
Type · 01
Supervised
Learn from labelled examples. A "label" is a human annotation — e.g., draw bounding boxes around cats so the model learns to spot them.
Type · 02
Unsupervised
Find patterns with no labels. The model groups data by similarity — no human tells it what to look for.

Example: clustering customers by shopping behaviour.
Type · 03 · LLMs
Self-Supervised
Creates its own training task from raw data — no labels needed. Example: hide words in a sentence, let the model predict the missing ones. Examples are created by the data itself.
LLMs are mainly self-supervised. "Large" = billions of parameters trained on nearly the entire internet.

Machine Learning — A Decade of Change
Classical vs
Deep.
Before 2012
Hand-Crafted Features
Linear Regression · Decision Trees · k-NN · SVM.

A human expert manually picks the properties ("features") the model uses. Works on small, structured data.
2012 → Now
Learns Its Own Features
No feature engineering — the network discovers what matters from raw data automatically.

The dominant approach today — the engine behind every LLM.

Deep Learning — Neural Networks
Neural
Networks.
Loosely inspired by the brain's neuron structure — nodes connected in layers. Each node represents a piece of useful information the network has learned.
→

→

"Deep" = many hidden layers. LLMs are transformer networks — billions of parameters, trained to predict the next word.
